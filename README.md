# PRESIDENT Database

เป็น Database ที่เก็บข้อมูลเกี่ยวกับประธานาธิบดีสหรัฐ ซึ่งนำมาใช้เป็นข้อมูลสำหรับหัดใช้งาน SQL Query ต่างๆได้

## วิธี RUN SQL SCRIPT กับ MYSQL

```sh

mysql -u root -p -h mysql_address < init.sql

```
